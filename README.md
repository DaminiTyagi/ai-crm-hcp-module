# ai-crm-hcp-module
AI-first CRM system using React, FastAPI, and LangGraph
## Overview
This project is an AI-first CRM system designed to manage Healthcare Professional (HCP) interactions using a conversational interface.

## Tech Stack
- Frontend: React
- Backend: FastAPI
- AI Framework: LangGraph
- LLM Ready: Groq integration supported

## Features
- Chat-based interaction logging
- Edit interaction
- Insight generation
- Suggested next actions
- Follow-up scheduling

## LangGraph Tools
1. Log Interaction
2. Edit Interaction
3. Insight Tool
4. Suggest Tool
5. Follow-up Tool

## Run Locally

### Backend
```bash
cd backend
python -m uvicorn app:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm start
```
